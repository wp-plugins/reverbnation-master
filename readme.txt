Reverbnation Master plugs-in perfectly into wordpress and allows you to display all the reverbnation juice inside any widget template position.

== Description ==

Great sound extension for wordpress is now available for download. Reverbnation is an award winning website and social community for independent music industry providing a central meeting point for musicians, producers and venues to communicate and collaborate.

Reverbnation Master plugin allows you to display all the reverbnation juicy widgets inside any wordpress widget template position. Yes, that's right, all great widgets in the palm of your hand, Html5 Player, Html5 Fan Collector, Html5 Show Schedule, Tune Widget, Shop, etc.

With fast loading speeds, Reverbnation Master as a very small system trace and is based in reverbnation html5 api. Check the screenshots page or get in touch with us in case of doubts.

IMPORTANT: Errors and conflicts free. NO JAVASCRIPT or AJAX.

**independent music for wordpress**

**LITE Version Contains:**

* Display or hide Widget Title
* Cool transparent Reverbnation Profile Button

**ADVANCED Version Contains:**

[Advanced Version Link](http://wordpress.techgasp.com/reverbnation-master/)

* Display or hide Widget Title
* Cool transparent Reverbnation Profile Button
* Reverbnation Widget Player (full nine yards api Player. Displays all Reverbnation Widgets, Html5 Player, Html5 Fan Collector, Html5 Show Schedule, Tune Widget, Shop, etc.)
* Shortcode Framework. Publish widget inside pages and posts

== Installation ==

1. Upload everything into the "/wp-content/plugins/" directory of your WordPress site.
2. Activate in the "Plugins" admin panel.
3. Visit the "Widget Page | Reverbnation Master" to configure the settings and your template widget position.

= Issues with buttons display =

= WHERE IS THE PLAYER? =
* ADVANCED Version contains Reverbnation Profile Button plus the Reverbnation Widget Player.

= Insert Widget in Pages or Posts =
* ADVANCED Version constains the shortcode framework
* Insert the code into the page or post. [reverbnationmaster id=x]
* "x" stands for your sidebar number, ie sidebar-1, final code would be:
* [reverbnationmaster id=sidebar-1]
